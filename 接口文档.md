## WebSocket 事件

### 客户端发送事件

#### 1. call_request - 发起通话请求

发起对目标用户的视频通话请求。

```json
{
  "target_sid": "目标用户的 Socket ID"
}
```

**错误响应**:
- `call_error`: 用户已离线 / 不能呼叫自己 / 对方正在通话中

---

#### 2. call_response - 响应通话请求

接听或拒绝来电。

```json
{
  "room_id": "房间ID",
  "accepted": true/false,
  "caller_sid": "呼叫方的 Socket ID"
}
```

**说明**:
- `accepted: true` - 接听通话
- `accepted: false` - 拒绝或取消通话

---

#### 3. join_room - 加入通话房间

接听后加入房间准备媒体交换。

```json
{
  "room_id": "房间ID"
}
```

---

#### 4. offer - 发送 WebRTC Offer

呼叫方发送 SDP Offer。

```json
{
  "room_id": "房间ID",
  "offer": "RTCSessionDescription"
}
```

---

#### 5. answer - 发送 WebRTC Answer

接听方发送 SDP Answer。

```json
{
  "room_id": "房间ID",
  "answer": "RTCSessionDescription"
}
```

---

#### 6. ice_candidate - 发送 ICE 候选

发送 ICE Candidate 用于 NAT 穿透。

```json
{
  "room_id": "房间ID",
  "candidate": "RTCIceCandidate"
}
```

---

#### 7. stream_ready - 媒体流就绪通知

通知对方本地媒体流已准备好。

```json
{
  "room_id": "房间ID",
  "type": "video/audio/both"
}
```

---

#### 8. end_call - 结束通话

挂断当前通话。

```json
{
  "room_id": "房间ID"
}
```

---

### 服务端发送事件

#### 1. user_assigned - 用户名分配

连接成功后分配随机用户名。

```json
{
  "username": "Call_ab12"
}
```

---

#### 2. user_list - 在线用户列表

广播当前所有在线用户。

```json
{
  "users": [
    {
      "username": "Call_ab12",
      "sid": "socket_id",
      "in_call": false
    }
  ]
}
```

---

#### 3. incoming_call - 来电通知

通知被呼叫方有来电。

```json
{
  "caller_sid": "呼叫方ID",
  "caller_name": "Call_ab12",
  "room_id": "room_xxx"
}
```

---

#### 4. call_ringing - 呼叫中通知

通知呼叫方正在等待对方接听。

```json
{
  "target_sid": "目标用户ID",
  "target_name": "Call_ab12",
  "room_id": "room_xxx"
}
```

---

#### 5. call_accepted - 通话已接听

双方收到通话建立通知。

```json
{
  "room_id": "房间ID",
  "peer_sid": "对方ID",
  "is_caller": true/false
}
```

---

#### 6. call_rejected - 通话被拒绝/取消

通话被拒绝或取消时通知双方。

```json
{
  "reason": "对方拒绝了通话"
}
```

**触发场景**:
- 被呼叫方拒绝接听
- 呼叫方取消呼叫
- 房间不存在（对方已断开）

---

#### 7. call_ended - 通话结束

对方挂断时通知。

```json
{
  "reason": "对方已挂断"
}
```

---

#### 8. call_error - 通话错误

通话请求出错时通知。

```json
{
  "message": "错误信息"
}
```

**错误类型**:
- 用户已离线
- 不能呼叫自己
- 对方正在通话中
- 通话已结束

---

#### 9. peer_joined - 对方加入房间

通知房间内其他用户有新用户加入。

```json
{
  "peer_sid": "对方ID",
  "username": "Call_ab12"
}
```

---

#### 10. offer / answer / ice_candidate - WebRTC 信令

转发 WebRTC 协商数据。

```json
// offer
{
  "offer": "RTCSessionDescription",
  "from_sid": "发送方ID",
  "from_name": "Call_ab12"
}

// answer
{
  "answer": "RTCSessionDescription",
  "from_sid": "发送方ID"
}

// ice_candidate
{
  "candidate": "RTCIceCandidate",
  "from_sid": "发送方ID"
}
```

---

#### 11. peer_stream_ready - 对方媒体流就绪

通知对方媒体流已准备好。

```json
{
  "peer_sid": "对方ID",
  "type": "video/audio/both"
}
```

## 数据结构

### 在线用户 (online_users)

```python
{
  "sid": {
    "username": "Call_ab12",
    "room": "room_id 或 None"
  }
}
```

### 房间信息 (rooms)

```python
{
  "room_id": {
    "users": ["sid1", "sid2"],
    "streams": {}
  }
}
```

### 用户媒体流 (user_streams)

```python
{
  "sid": {
    "video": true/false,
    "audio": true/false
  }
}
```

## 通话流程

```
┌─────────┐                    ┌─────────┐
│  呼叫方  │                    │ 被呼叫方 │
└────┬────┘                    └────┬────┘
     │                              │
     │  1. call_request            │
     │ ───────────────────────────>│
     │                              │
     │  2. call_ringing            │
     │ <───────────────────────────│
     │                              │
     │  3. incoming_call           │
     │                              │<── 显示来电弹窗
     │                              │
     │  4. call_response(accepted) │
     │ <───────────────────────────│
     │                              │
     │  5. call_accepted           │
     │ <──────────────────────────>│
     │                              │
     │  6. join_room               │
     │ <──────────────────────────>│
     │                              │
     │  7. offer/answer/ice        │
     │ <──────────────────────────>│
     │                              │
     │  8. end_call                │
     │ <──────────────────────────>│
     │                              │
```

## 错误码

| 错误信息 | 说明 |
|---------|------|
| 用户已离线 | 目标用户已断开连接 |
| 不能呼叫自己 | 无法向自己发起通话 |
| 对方正在通话中 | 目标用户已在其他通话中 |
| 通话已结束 | 房间已被清理 |
| 对方拒绝了通话 | 对方主动拒绝或取消 |
| 对方已挂断 | 通话中对方挂断 |
